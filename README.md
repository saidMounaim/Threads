## Features:

- Login with GitHub or Google
- Make and Delete Threads
- Like and Comment on Threads
- Follow or Unfollow Users

## Technology Stack:

- Next.js
- Next-Auth
- Prisma
- TypeScript
- TailwindCSS
- Shadcn UI

## Usage

### Env Variables

Create a .env file in then root and add the following

```
DATABASE_URL=

GITHUB_ID=
GITHUB_SECRET=

GOOGLE_CLIENT_ID=
GOOGLE_CLIENT_SECRET=
```

## Install Dependencies

```
npm install
```

### Run

```
# Run project
npm run dev
```
